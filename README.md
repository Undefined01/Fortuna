#### 使用方法

该成绩查询系统分为前端和后端。

使用时请在[Releases](https://github.com/Undefined01/fortuna/releases)中下载发布的版本。

解压后请启动`backend.exe`，然后使用浏览器访问[本地的8081端口](http://localhost:8081)即可。

（或使用 `electron` 作为程序启动器）
